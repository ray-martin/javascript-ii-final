Content Outline and Competencies:

  1. DOM
    1. Describe the DOM.
    2. Modify HTML elements.
    3. Open and close windows and tabs.
    4. Utilize the History, Location, Navigator and Screen objects.
  2. HTML Document Nodes
    1. Add document nodes.
    2. Remove document nodes.
  3. BOM
    1. Open and close windows.
    2. Access with the History object.
    3. Use the Location object.
    4. Use the Navigator object.
    5. Manipulate the Screen object.
  4. Forms
    1. Improve form usability.
    2. Program custom form validation.
    3. Identify data passed through a query string.
  5. JavaScript Object-Oriented Programming
    1. Explain the principles of object-oriented programming.
    2. Create and use objects using the Date class.
    3. Create and use objects using the Number class.
    4. Create and use objects using the Math class.
    5. Define custom objects.
  6. Strings and Arrays
    1. Find, extract, combine and compare strings.
    2. Code regular expressions.
    3. Find, extract, sort and combine arrays.
  7. Validation and Debugging
    1. Use lint to improve and fix code.
    2. Write error handling code.
    3. Use debugger windows from all major browsers.
  8. Security
    1. Write code to store state information.
      1. Query strings
      2. Hidden form fields
    2. Investigate and apply current security holes and techniques.
